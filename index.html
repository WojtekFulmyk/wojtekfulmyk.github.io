My Projects


  • Nonlinear Granger Causality using Kernel Ridge Regression

    In this paper I show that the mlcausality method with kernel ridge regression outperforms rival nonlinear Granger causality algorithms. Greater accuracy is acheived at execution speeds that are 10+ times lower than competing methods.

  • mlcausality - time series analysis using ML

    I develop mlcausality, a Python library for identifying nonlinear Granger causal relationships using machine learning techniques. This library allows the user to use many different machine learning techniques, such as random forests, XGBoost, or SVR, to recover nonlinear Granger causal relationships.

     code

  • ICR - Identify Age-Related Conditions

    Top 3% finish on a highly competitive Kaggle competition. Used an ensemble of 25 CatBoost models to identify whether an age-related condition occurs from data with anonymized features. Used a repeated stratified K-fold cross-validation strategy to tune CatBoost hyperparameters.

     solution writeup solution code leaderboard position

  • AMP - Parkinson's Disease Progression

    43rd place (top 3% finish) on a challenging Kaggle competition. The objective was to predict Parkinson's disease progression on 4 separate scoring criteria at 4 separate points in the future using protein and peptide data. My solution used a combination of numerous CatBoost and linear regression models with parameters tuned using GroupKFold cross-validation.

    solution writeup solution code leaderboard position

  • Feedback Prize - English Language Learning

    24th place (top 4%) on the efficiency leaderboard on a competitive Kaggle competition. The objective of the competition was to predict essay scores on 6 different metrics. My solution involved extracting embeddings from the DeBERTa model and predicting essay scores using SVR and CatBoost. This solution achieved excellent results without fine-tuning DeBERTa!
